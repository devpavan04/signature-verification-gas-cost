# Signature verification gas cost

Returns the gas cost associated with signature verification on a contract.

## quickstart

```bash
git clone https://github.com/devpavan04/signature-verification-gas-cost
cd signature-verification-gas-cost
npm install
npm run chain
```

> in a second terminal window:

```bash
npm run deploy
cd client
npm install
npm start
```
